Welcome to English to Telugu Rule based Machine Translation System.

To setup the following you need to install following before running

1.) Java JDK, JRE latest versions.
2.) Install XAMPP server.
3.) Install Mysql connector j for java. and setup path and classpath
4.) After installing Xampp, run MySQL and Appache Modules in Xamppp  control panel
5.) Open localhost/phpmyadmin and goto import options.
6.) Create database nlp and import nlp.sql file

Steps to be followed to run the program

1.) Run Mysql and Apache modules in Xampp control panel.
2.) Compile the program in command prompt if not compiled.
3.) Run the program.
4.) Enter the English text and click translate.
